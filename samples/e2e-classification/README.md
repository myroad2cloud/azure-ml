# End-to-End Classification Sample

## Purpose

This sample provides a small runnable Azure ML project structure.

It includes:

- Training script
- Scoring script
- Environment definition
- Azure ML job definition
- Managed online endpoint placeholders
- Smoke test client

## Flow

1. Train model
2. Save model artifact
3. Register model
4. Deploy endpoint
5. Run smoke test

## Notes

This is a starter sample. Replace the toy dataset with a real dataset for production learning.
