# Enterprise Azure ML DevOps Playbook

This guide teaches Azure DevOps and Azure Machine Learning from an enterprise delivery angle.

It is built for architects who need to design, explain, and implement secure CI/CD for Azure Machine Learning in customer environments.

## What this playbook covers

- Azure DevOps organization and project setup
- Azure Repos and branching strategy
- Azure Resource Manager service connections
- Workload identity federation
- Self-hosted Azure DevOps agents
- Private Endpoints and Private DNS
- Secure Azure Machine Learning workspace design
- CI pipeline for validation and packaging
- CD pipeline for training, model registration, and deployment
- Infrastructure as Code using Terraform and Bicep
- Security, governance, monitoring, and troubleshooting

## Recommended learning order

1. Azure DevOps foundations
2. Git and branching
3. Service connections and identity
4. Self-hosted agents
5. Private networking
6. Azure ML workspace
7. CI pipeline
8. CD pipeline
9. MLOps lifecycle
10. Security and operations
